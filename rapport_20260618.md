1. Ouvrir les paramètres de l'environnement sur [code.claude.com](https://code.claude.com)
2. Accéder à la section **Network egress settings**
3. Ajouter le domaine : `bodacc-datadila.opendatasoft.com`
4. Documenter : [Claude Code on the web — environment configuration](https://code.claude.com/docs/en/claude-code-on-the-web)
